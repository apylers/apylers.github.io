Nginx 升级到最新 Stable 版本

2019-10-13 · 1 分钟 · 154 字 · awsl

Ubuntu 中直接安装的 Nginx 并非最新的 Stable 版本，在这里本教程参考 Nginx 官方教程 给出的方法，教大家如何升级。

1
2
3
4
5
6
7

$ apt install curl gnupg2 ca-certificates lsb-release  # 安装必需组件
$ echo "deb http://nginx.org/packages/ubuntu `lsb_release -cs` nginx" | tee /etc/apt/sources.list.d/nginx.list  # 安装 Nginx 官方 Stable 源
$ curl -fsSL https://nginx.org/keys/nginx_signing.key | sudo apt-key add -  # 安装 Nginx 签名用于 apt 验证
$ apt-key fingerprint ABF5BD827BD9BF62  # 验证签名
pub   2048R/7BD9BF62 2011-08-19 [expires: 2024-06-14]
      Key fingerprint = 573B FD6B 3D8F BC64 1079  A6AB ABF5 BD82 7BD9 BF62
uid                  nginx signing key <signing-key@nginx.com>

然后正常安装 Nginx 即可：

1
2

$ apt update
$ apt install nginx
